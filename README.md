# Proyecto-1-Irina-Spagnuolo
El siguiente proyecto tiene como proposito modelar mediante un grafo, las relaciones que hay entre los usuarios en una red social. El programa permite al usuario cargar una lista de usuarios y sus respectvas conexiones, asi como tambien le permite agregar nuevos usuarios y conexiones, y eliminar usuarios relizando inmediatamente la eliminaci√≥n tambien de las conexiones que dicho usuario tenia. Estos cambios el usuario los puede actualizar en el archivo de texton que habia sido cargado. El programa implementa el algoritmo Kasaraju el cual permite ver que tan fuerte es la conexion de los nodos, que traduciendolo a el proyecto seria, quien sigue a quien; si ambos usuarios se siguen los nodos apareceran en verde, si solo un usuario sigue al otro apareceran el naranja, y si quedan usuarios que no siguen a nadie y nadie los sigue a ellos, estos apareceran en gris.
